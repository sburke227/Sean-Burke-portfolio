Problems I encountered

The first problem I encountered was when I was reading the brief when I was looking at basic requirements number 2. I did not fully understand data validation. To help me with this issue I used the Golden Keys Computer Science Book(Ref 5) and studied chapter 3(Analytics) . I was able to improve my knowledge on data validation and improve my project. The next problem I encountered was with my embedded system. I attempted to create a stopwatch but instead created a timer. What I mean by this is on the input screen down the timer would begin to count but when the on screen up input came after the time would continue to go up without resetting. To fix this problem I added an input for on button b pressed which will set the variable I created to 0 allowing the timer to reset. The final problem I encountered was when I was completing my unit testing. While completing the function unitTest() and unitTest2(), the output of the test was always the same “Test result: Failed” regardless of the inputted data. I tried to change the conditions within the function but nothing worked. I then found out about global variables from W3Schools(Ref 6). By using a global key it allowed me to take the variable from outside the function and change it inside the function. This helped me to allow the function to output “Test result: True”.
